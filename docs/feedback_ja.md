
# Day 10 YouTube ライブ配信テスト フィードバック

### 1. **全レス問題（過負荷・ラグ）**
*   全てのコメントを1つずつ拾おうとしており、コメントが増えると処理が追いつかなくなる。
*   結果として、ラグが発生し、リアルタイム性が失われる。
*   **改善案**: 全部拾うのではなく、ランダムに拾う、あるいは重要なものを優先するなどの間引き処理が必要。

### 2. **レスポンス速度（Latency）**
*   コメント入力から読み上げ（反応）までのラグがまだ気になる。
*   パフォーマンス改善が必要。

### 3. **会話の単発性（Context）**
*   一問一答で終わってしまい、次のコメントに行くと前の話を忘れているように見える。
*   人間味が薄い。
*   **改善案**: 1つのコメントに対して話を膨らませる、あるいはランダムに拾ったコメントに対して深掘りするなど、「人間らしい会話の継続性」を持たせる。

### 4. **話題の散乱（Topic Drift）**
*   雑談のテーマを決めていなかったため仕方ない部分もあるが、来たコメントに振り回されすぎて話題が定まらない。
*   **改善案**: テーマからブレないように制御する、あるいは「雑談」としての芯を持たせるロジックが必要。
